## GitHub commands:
### Clone:
```
$ git clone [repo_url]
```
### Set Up:
```
$ git status
$ git add .
$ git status
```
### Commit:
```
$ git commit -m "Message"
```
### Push
```
$ git push origin main
```

## C compiling:
```
$ gcc file.c -o file       (compile)
$ ./file                   (execute)
```
It some cases, it is necessary to include certain libraries.
```
$ gcc file.c -o file -l{library name}
```
